# Security Policy
Pull Requests with unneccesary modifications in system commands like `renice` will not be accepted unless important.
## Supported Versions

v3.1.0 and above supports pull requests as of 6th April 2026. All other pull requests for lower versions will not be accepted

| Version | Supported          |
| ------- | ------------------ |
| 3.1.0   | :white_check_mark: |
| 3.2.0   | :white_check_mark: |


## Reporting a Vulnerability

Feel free to report vulnerabilities 🙂.
